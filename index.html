Presleyn

LunaSim

LunaSim is a system dynamics modelling website that aimed to replace propriety software for 9th grade SMCS. I'm actively working on improving its design to make it more user-friendly and run smoothly for future years of SMCS students.

Rapidfire

A retention-focused review tool with a focus on raw efficiency, Rapidfire is a no-strings-attached flashcard site that I built to help myself and others practice & study for numerous AP courses.
